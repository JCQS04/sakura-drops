jQuery.fn.toolbar(options)
Plugin to create custom ui-related bindings for a toolbar control. Uses the pattern of lazy-initializing the API and storing an instance of it as a data reference to the jQuery object. 
$toolbar = jQuery('#the-toolbar').toolbar(); // get the lazy-constructed API
$toolbar.hideButton($('#some-button', $toolbar)); // hide the button
Parameters:
{?Object=} options
Custom overrides to the defaults.
Returns:
{jQuery} Returns an extended jQuery object with the toolbar API methods.
